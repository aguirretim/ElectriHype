# ElectriHype

A simple social media Android app where users can create posts, follow others, and interact in a feed.

## Features
- User posts and feed
- Follow / unfollow users
- Clean social feed UI

## Tech Stack
- **Language**: Java
- **Platform**: Android
- **Build**: Gradle / Android Studio

## Getting Started
1. Clone the repo
2. Open in Android Studio
3. Build and run on an emulator or device
